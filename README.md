# Jogo da Memória com LED - Arduino

## Documentação do Projeto

Este projeto é um jogo da memória utilizando Arduino, LEDs, botões e buzzer. Ele foi desenvolvido para o seminário da disciplina de Sistemas Embarcados.

### Objetivo

Criar um jogo interativo onde o jogador deve repetir sequências de luzes e sons, testando sua memória e atenção.

### Funcionamento

- O Arduino acende uma sequência de LEDs e toca sons correspondentes.
- O jogador deve repetir a sequência pressionando os botões corretos.
- O jogo possui modos de memória (um jogador) e batalha (dois jogadores).
- Um easter egg musical pode ser ativado pressionando o botão amarelo ao ligar.

### Como Usar

1. Monte o circuito conforme o esquema.
2. Carregue o código no Arduino.
3. Siga as instruções dos LEDs e sons para jogar.
